# 🏠 Hostel Management System - UOR

## 📖 Overview

The Hostel Management System is a web-based application developed to streamline hostel allocation and room management processes at a university. The system enables Student Affairs officers to manage hostels, rooms, and student allocations efficiently while reducing manual paperwork and improving data accuracy.

This project is being developed as an internship project to provide a centralized platform for hostel administration.

---

## 🎯 Objectives

- Simplify hostel allocation processes.
- Manage hostel and room information efficiently.
- Maintain student accommodation records.
- Reduce manual administrative work.
- Improve transparency in room assignments.

---

## 🚀 Features

### 👨‍🎓 Student Management
- View student information.
- Search students by registration number.
- Manage student hostel allocations.

### 🏢 Hostel Management
- Add new hostels.
- Update hostel information.
- View available hostels.
- Manage hostel capacities.

### 🚪 Room Management
- Add rooms to hostels.
- Update room details.
- View room availability.
- Track occupied and vacant rooms.

### 📋 Allocation Management
- Allocate students to rooms.
- View allocation history.
- Manage room assignments.
- Prevent over-allocation.

### 👨‍💼 Officer Management
- Student Affairs Officer login.
- Secure access to management functions.
- Role-based system operations.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/HostelManagementSystem-UOR.git
```

### Backend Setup

```bash
cd backend
mvn spring-boot:run
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
